LeetCode 462 最少移动次数使数组元素相等 II

minimum moves to equal array elements ii

题目

给定一个非空整数数组，找到使所有数组元素相等所需的最小移动数，其中每次移动可将选定的一个元素加 1 或减 1。 您可以假设数组的长度最多为 10000。

例如:

输入:
[1,2,3]

输出:
2

说明：只有两个动作是必要的（记得每一步仅可使其中一个元素加 1 或减 1）：

[1,2,3] => [2,2,3] => [2,2,2]

思路

举个栗子 , 对于数组 [0, 0, 1, 7, 8] 来说, 第一想法肯定是取中间的数例如取 4 ,那么此时各个位置的差值为 4 4 3 3 4
如果取 3 ,那么此时各个位置的差值为 3 3 2 4 5

也就是当取值减 1 时, 比它小的数的差值会减一, 比它大的数的差值就会加一所以如果比它小的数的个数比比它大的数的个数多, 那我们就是赚的那什么时候可以停止呢? 肯定是比它小的数比比它大的数的个数不多的时候

因此我们取的值应当是居中的一个中位数值可以测试得知, 如果数组长度为偶数, 取中间两个数任意一个都可以

因此对于长度为 len 的数组而言, 取 Math.floor(len/2) 即可
