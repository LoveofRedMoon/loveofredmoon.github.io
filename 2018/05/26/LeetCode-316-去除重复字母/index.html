LeetCode 316 去除重复字母

题目

给定一个仅包含小写字母的字符串，去除重复的字母使得所有字母出现且仅出现一次。你必须保证返回结果是所有可能结果中的以字典排序的最短结果。

例如:

给定 “bcabc”
返回 “abc”

给定 “cbacdcbc”
返回 “acdb”

思路

分别统计各个字母出现的位置

如果按照字典序, 因此我们寻找位置应当从 a->z 进行寻找

从前一个字符的位置向后寻找对应字符例如cbacdcbc

  1. cbacdcbc
  2. 再 cbacdcbc
  3. 再 cbacdcbc
  4. 发现 d 找不到了, 但是 d 必须存在, 因此直接找到最后一个d所在位置

这时生成了一个相对小的字典排序 adbc 但是不是最小的这时哪些字母的相对位置一定确定了呢?
